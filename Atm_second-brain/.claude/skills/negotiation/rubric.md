# Negotiation Readiness Rubric (binary — every box must be checked before going live)

- [ ] BATNA is identified, specific, and (if weak) actively improved before negotiating.
- [ ] Reservation point (walk-away number/terms) is set as a concrete value.
- [ ] Target point is set: ambitious but defensible with a stated rationale.
- [ ] Counterparty's likely BATNA and reservation point are estimated.
- [ ] ZOPA is assessed; if none exists, the issue set is reframed or the walk decision is made.
- [ ] Interests (the "why") are listed separately from positions (the "what") for both sides.
- [ ] At least one value-creating trade across differently-valued issues is identified.
- [ ] Anchor decision is made: who opens first, at what number, with what justification.
- [ ] Every planned concession is paired with an explicit "in return for" (no unilateral gives).
- [ ] Concessions are sequenced in decreasing size to signal approach to the floor.
- [ ] Roles/process are set: who speaks, who has authority to commit, agenda, package vs sequential.
- [ ] Deadline reality and both sides' authority limits are confirmed (not assumed).
- [ ] At least three calibrated open questions are prepared to surface interests.
- [ ] A closing checklist exists: summarize terms, confirm authority, get it in writing.
- [ ] `personal/` recall has been consulted; any conflict with generic advice is flagged and personal/ wins.
