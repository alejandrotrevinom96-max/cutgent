# Anti-patterns — Health & Fitness

## Program design
- **"Muscle confusion" / random workouts.** No tracked progression. *Fix:* pick
  movements and progress them.
- **Once-per-muscle bro-split for novice/intermediate.** *Fix:* aim ~2x/week.
- **All isolation, no compounds.** *Fix:* anchor in compounds.
- **Junk volume / ego-overreach.** 30+ sets, maxing every session. *Fix:* MEV->MAV, deload.
- **Program-hopping.** New program every 2-3 weeks. *Fix:* run a plan 6-12 weeks, then assess.
- **No deload, ever.** *Fix:* schedule deloads; back off on warning signs.

## Nutrition
- **Calories ignored, "clean eating" only.** *Fix:* set surplus/deficit/maintenance.
- **Protein too low.** *Fix:* set a target (~1.6 g/kg+).
- **Crash diet / dirty bulk.** *Fix:* moderate (~+-10-20%).
- **Supplement-first thinking.** *Fix:* fix sleep/calories/protein/training first.

## Recovery
- **Sleep sacrificed while adding volume.** *Fix:* fix sleep first.
- **"No days off."** *Fix:* schedule rest.

## Safety / scope (most important)
- **Training through pain.** *Fix:* stop, regress load, refer to a professional.
- **Playing doctor.** Diagnosing/prescribing for a medical issue. *Fix:* refuse;
  route to a physician/PT/RD; keep guidance general.
- **No disclaimer.** *Fix:* add it.
- **Stale "official" numbers.** *Fix:* date-stamp guideline specifics.
