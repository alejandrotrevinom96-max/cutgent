# Legal-literacy Rubric (binary — every box must be checked)

- [ ] The "general education, NOT legal advice" disclaimer is surfaced prominently in the response.
- [ ] The governing jurisdiction (country + state/province) is identified or explicitly asked for; it is stated that the answer can vary by jurisdiction.
- [ ] The question is classified into a bucket: contract / IP / licensing / entity / compliance.
- [ ] No legal conclusion is asserted as certain; uncertainty is marked plainly.
- [ ] For high-stakes matters (signing, disputes, threats, large money, irreversible rights), a "consult a licensed attorney" trigger is explicitly named.
- [ ] For a contract, the key elements are checked: parties, scope, payment, IP ownership vs license, termination, liability/indemnity, warranties, governing law.
- [ ] IP ownership (assignment / work-for-hire) is distinguished from a license, and any clause transferring the user's IP is flagged.
- [ ] For IP questions, the correct regime (copyright vs trademark vs patent) is named rather than conflated.
- [ ] Fair use / fair dealing, if raised, is treated as narrow and fact-specific, not as a blanket right.
- [ ] For a license, scope is pinned down: exclusive/non-exclusive, territory, term, and media/use.
- [ ] For content, the needed releases/clearances (talent, location, music/footage/image/font) are identified.
- [ ] Dangerous clauses (broad IP assignment, uncapped indemnity, perpetual license, non-compete, personal guarantee, auto-renew) are flagged where present.
- [ ] `personal/` recall has been consulted; any conflict with generic info is flagged and personal/ wins.
