# Anti-patterns — Psychology

## Pop-psych stated as fact
"We use 10% of our brains," "left/right-brained." *Fix:* cite robust findings; flag
contested ones as contested.

## Pathologizing / armchair diagnosis
"That's OCD," "he's a narcissist," "you're traumatized" on ordinary behavior. *Fix:*
describe behavior, not diagnoses; route real concerns to a professional.

## MBTI / enneagram as truth
"You're an INTJ, so..." MBTI is **not validated** (poor test-retest, no clean
types). *Fix:* use validated dimensional constructs at most, held loosely.

## Willpower-only habit advice
"Just be more disciplined." *Fix:* make it easier, fix the cue, design the environment.

## Ignoring the environment
Treating behavior as pure character while the situation does the work. *Fix:* change
the context first; guard against the fundamental attribution error.

## Manipulation via influence tactics
Using Cialdini to pressure, fabricate scarcity/social proof, or exploit. *Fix:* only
true principles to support a genuine decision.

## Single-study overreach
Generalizing one small/unreplicated study into a law. *Fix:* prefer meta-analytic,
well-replicated evidence; one study is a hypothesis.

## Ignoring effect size
Treating "statistically significant" as "big and important." *Fix:* report magnitude
and uncertainty, not just the label.
