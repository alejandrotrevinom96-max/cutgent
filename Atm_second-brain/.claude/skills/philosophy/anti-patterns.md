# Philosophy anti-patterns

## Dogmatism / preaching one school
*Tell:* the answer is delivered as "Stoicism teaches…" as settled. *Fix:* apply
multiple lenses, hold the conclusion provisionally, let the user's values weigh in.

## Strawmanning the opposition
*Tell:* the rejected view is stated weakly. *Fix:* steelman first; engage the best version.

## Conflating validity with truth
*Tell:* "the argument is valid, therefore true." *Fix:* validity is form (if
premises, then conclusion); truth needs sound premises. Check both, separately.

## Ignoring the is-ought gap
*Tell:* sliding from "this is natural / how it is" to "how it *should* be" with no
value premise. *Fix:* name the missing normative premise and assess it.

## Motivated reasoning
*Tell:* every argument conveniently supports a desired conclusion. *Fix:* ask "what
would change my mind?", argue the other side, separate true from comfortable.

## Fallacies (false dilemma & friends)
*Tell:* "either X or Y" when more options exist; ad hominem; equivocation; appeal to
authority as proof; circularity. *Fix:* name the fallacy and repair the argument.

## Obscurantist jargon
*Tell:* complexity used to impress, no plain-language anchor. *Fix:* define every
term in ordinary words; if it can't be said plainly, it may not be understood.

## Treating normative as empirical (or vice versa)
*Tell:* "the data will tell us whether this is *right*"; or endless conceptual debate
over a factual matter. *Fix:* classify the question type first and match the tools.
