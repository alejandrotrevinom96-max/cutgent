# Anti-patterns — learning

## Rereading and highlighting
Familiarity isn't retrieval ability. **Fix:** close the book and recall; turn review
into self-testing.

## Massed practice / cramming
Inflates short-term performance, collapses long-term retention. **Fix:** space
sessions; let some forgetting happen.

## Blocking instead of interleaving
You learn to execute a known procedure, not to choose it; transfer is poor. **Fix:**
mix types within a session even though it feels worse.

## Practicing what you're already good at
No edge → no growth; that's performance, not practice. **Fix:** drill the sub-skill
you fail at.

## Passive input as a stand-in for practice
Consumption isn't production. **Fix:** directness — produce the thing early; learn
from the gaps.

## Illusion of competence
"I get it" after watching the solution. **Fix:** Feynman it from a blank page; if
you stall, you don't have it.

## No feedback loop
Reps without knowing if they're right groove in errors. **Fix:** install an answer
key/tests/critic before volume.

## Anki abuse
Verbatim, multi-fact, downloaded cards; skipped reviews. **Fix:** one atomic fact
per card in your own words from understood material; review daily.

## Tweaking the system instead of doing the work
Reorganizing notes, app-shopping, watching study-method videos. **Fix:** cap setup
time; the rubric decides if it's working.

## Collecting notes you never revisit or test
Storage isn't memory. **Fix:** encode atomic evergreen notes, link them, convert
testable atoms to cards.
