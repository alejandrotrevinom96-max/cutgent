# Rubric — copywriting (binary pass/fail)

- [ ] **One action** — the copy asks for exactly one thing; the CTA is explicit.
- [ ] **Reader-centered** — speaks to the reader's desire/pain, not the company.
- [ ] **Benefit-led** — leads with the outcome; features support, don't open.
- [ ] **Specific** — concrete numbers/nouns/proof; no empty superlatives.
- [ ] **Headline earns the read** — clear promise or curiosity in <12 words; a
      reader who reads only the headline knows the offer.
- [ ] **Flow** — each line makes you want the next (no dead sentences).
- [ ] **Proof present** — at least one credible proof element near the claim.
- [ ] **Friction removed** — risk reducers at the CTA (guarantee/social proof).
- [ ] **Tight** — no hedges, filler, or redundant words; survives read-aloud.
- [ ] **Voice match** — consistent with the brand's documented tone.
- [ ] **Honest** — every claim is true and defensible; scarcity is real.
