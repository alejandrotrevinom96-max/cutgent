# Sources

## Canonical (stable craft)
- Richard Williams — *The Animator's Survival Kit*. Timing, spacing, weight,
  mechanics of motion.
- Frank Thomas & Ollie Johnston — *The Illusion of Life: Disney Animation*. Origin
  of the 12 principles.
- Jeremy Birn — *Digital Lighting & Rendering*. Three-point, motivated light, exposure.
- Physically Based Rendering literature for PBR theory.
- Gnomon Workshop; Animation Mentor / iAnimate (blocking → spline → polish).

## Tool & vendor guides (DATE-STAMPED — fast-moving; verify current version)
- Blender Manual & Blender Studio training (as of 2026-06; ~3 releases/year).
- Adobe Substance 3D PBR Guide — canonical metal/rough value reference (as of 2026-06).
- Autodesk Maya / Maxon C4D / SideFX Houdini official docs — version-pinned.
- Poly Haven — free HDRIs/textures/assets for look-dev (as of 2026-06).

## Note on generative/AI 3D (fast-moving — verify capability claims)
Treat current text/image-to-3D and scan tools as concept/dressing/raw-stock
generators. As of 2026-06, none reliably produce clean deforming quad topology or
art-directed performance — re-verify before relying on any specific tool for
production hero assets.
