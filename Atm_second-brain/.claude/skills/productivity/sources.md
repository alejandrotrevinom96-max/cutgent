# Sources — productivity

## Books
- Allen — *Getting Things Done* (2001; rev. 2015). Capture/clarify/organize/reflect/
  engage; next-actions; 2-minute rule; weekly review.
- Newport — *Deep Work* (2016). Deep vs shallow; scheduling/ritualizing focus.
- Newport — *Slow Productivity* (2024). Do fewer things, natural pace, quality.
- Forte — *Building a Second Brain* (2022). CODE and the PARA method.
- Covey — *The 7 Habits* (1989). "Put first things first"; the importance/urgency
  matrix (Eisenhower matrix, Quadrant II focus).

## Concepts
- Eisenhower matrix — Eisenhower's urgent/important distinction, popularized by Covey.
- Timeboxing/timeblocking — calendar-driven scheduling (Newport; Forster).
- Pomodoro — Cirillo; timed focus intervals with breaks.

## Honesty note
These are practitioner frameworks, not controlled science. The robust core:
**single-tasking beats multitasking** (switching costs are documented) and
**externalizing commitments reduces cognitive load**. The specific rituals are
useful scaffolding, not proven formulas — keep what moves the user's goals. The
personal layer overrides any framework here.
