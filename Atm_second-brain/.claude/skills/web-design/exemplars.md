# Exemplars — web design

- **Stripe** — dense information made calm through hierarchy, generous space, and
  a restrained palette with one decisive accent. *Lesson:* clarity scales.
- **Linear** — opinionated type, precise spacing, motion that explains state.
  *Lesson:* craft in micro-details signals product quality.
- **Apple product pages** — one message per scroll section, huge imagery, minimal
  copy. *Lesson:* pacing and focus beat density.
- **Awwwards "Site of the Day" winners** — study the *grid and motion*, not the
  decoration; the best ones have a single strong idea executed precisely.
- **Refactoring UI patterns** — start with too much space and remove; use color
  and weight (not borders) to separate. *Lesson:* defaults are the enemy.

Use: name the closest exemplar's *principle* (hierarchy, pacing, restraint) and
design toward it — never copy the surface.
