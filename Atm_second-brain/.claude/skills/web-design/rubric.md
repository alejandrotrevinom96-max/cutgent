# Rubric — web design (binary pass/fail)

- [ ] **One clear job** — the primary action/message is obvious within 3 seconds.
- [ ] **Hierarchy** — squint test: the order of attention is unambiguous.
- [ ] **Type scale** — a real scale (not arbitrary px); body line-length 45–75ch;
      line-height ~1.4–1.6; ≤ 2 font families.
- [ ] **Spacing system** — all spacing maps to a 4/8px scale; grouping uses space,
      not boxes-everywhere.
- [ ] **Contrast** — text meets WCAG AA (4.5:1 body, 3:1 large).
- [ ] **Color discipline** — one accent, restrained neutrals; accent reserved for
      the primary action.
- [ ] **Alignment** — everything sits on a grid; no off-by-a-few-pixels drift.
- [ ] **Responsive** — works at 360px and at 1440px+; tap targets ≥ 44px.
- [ ] **Motion** — purposeful, 150–300ms, eased; respects reduced-motion.
- [ ] **States** — hover/focus/active/empty/error/loading all designed.
- [ ] **Performance budget** — no layout shift; images sized; fonts not blocking.
- [ ] **Accessibility** — semantic structure, visible focus, labelled controls.
