# Sources — web design

- *Refactoring UI* — Adam Wathan & Steve Schoger (practical hierarchy, spacing,
  color).
- *The Elements of Typographic Style* — Robert Bringhurst (typography canon).
- *Don't Make Me Think* — Steve Krug (usability/clarity).
- *Grid Systems in Graphic Design* — Josef Müller-Brockmann (grids).
- *Refactoring UI* + *Practical Typography* (Butterick) for type detail.
- **WCAG 2.2** — accessibility/contrast requirements (authoritative; verify the
  current version).
- Pattern references: Awwwards, Godly, Mobbin (study principles, not skins).
- Performance: web.dev / Core Web Vitals (treat as living docs; date-stamp).
