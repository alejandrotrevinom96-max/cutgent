# Rubric — image & video (binary pass/fail)

Ship only when every applicable check is a clear **pass**. A single fail = iterate
with the matching edit op, not a full re-roll.

## Image

- [ ] **Brief match** — subject, style, and intended use are unmistakably met.
- [ ] **Aspect/resolution** — correct aspect ratio for the destination, and
      resolution ≥ the platform's requirement (upscale if short).
- [ ] **Anatomy/structure** — no malformed hands, teeth, eyes, or duplicated
      limbs; text in-image is spelled correctly or absent by intent.
- [ ] **Focal clarity** — the subject reads in <1s; the eye lands where intended.
- [ ] **Lighting coherence** — one consistent light logic; shadows match the key.
- [ ] **Edges/compositing** — cutouts have clean edges, no halo or fringe.
- [ ] **No artifacts** — no warping, smearing, repeating texture, or watermark.
- [ ] **Color** — intentional palette; correct white balance; no clipping.

## Video (adds to the above, per representative frames)

- [ ] **Temporal stability** — no flicker, identity drift, or popping between
      frames; the subject stays itself.
- [ ] **Motion plausibility** — movement obeys physics/weight; no sliding feet,
      no impossible morphs.
- [ ] **Hook** — the first 1–2 seconds earn the next five (for social).
- [ ] **Audio sync** — if dubbed/voiced, lips and beats line up.
- [ ] **Cut rhythm** — pacing matches the platform and the message.

## Always

- [ ] **Provenance recorded** — prompt, model, seed/params saved with the asset.
- [ ] **Rights/consent** — no protected likeness or IP used without basis.
