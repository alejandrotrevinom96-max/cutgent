# Photography Rubric (binary — every box must pass before a frame ships)

- [ ] The subject is identifiable and there is a single stated intent/idea the frame serves.
- [ ] Light was assessed for quality, direction, color, and ratio — and chosen, not accepted by default.
- [ ] The aperture is set for the intended depth of field (subject isolation vs front-to-back sharpness).
- [ ] The shutter speed is set for the intended motion rendering (freeze vs blur) and is fast enough to avoid unintended shake.
- [ ] ISO is as low as the scene allows after aperture and shutter are committed.
- [ ] Exposure is confirmed on the histogram; no unrecoverable clipped highlights (unless deliberate).
- [ ] The frame is simplified: everything not serving the idea is removed or hidden.
- [ ] A deliberate compositional structure is present (thirds/balance, leading lines, frame-in-frame, or intentional symmetry) — not accidental centering.
- [ ] Edges, corners, and background are checked for mergers, bright distractions, and clutter.
- [ ] Focal length and working distance were chosen for the intended perspective/compression — perspective set by foot position, framing by lens.
- [ ] Focus is on the intended point (eyes for people) and depth of field actually covers what must be sharp.
- [ ] For people: light, angle, and lens are flattering, and pose/direction create natural angles, space, and genuine expression.
- [ ] For time-sensitive subjects, the moment/peak gesture was anticipated and captured (not a random frame).
- [ ] The edit ran in order — cull, then global before local — with white balance and tone set before color and retouch; retouch keeps texture real; output is sharpened for its medium.
- [ ] In documentary contexts, manipulation stays within disclosed limits, and subject consent/releases and privacy are respected.
- [ ] `personal/` recall has been consulted (the user's gear, style, subjects, presets); any conflict with generic advice is flagged and personal/ wins.
