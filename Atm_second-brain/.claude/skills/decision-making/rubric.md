# Decision-Making Rubric (BINARY)

Scale effort to the door: a clear two-way door may *pass by deciding fast* (over-
deliberation is itself a fail). For a one-way door, every box must be genuinely checked.

- [ ] The actual decision and the options are written down explicitly.
- [ ] Reversibility is classified: one-way door (irreversible) vs two-way door (reversible).
- [ ] Effort is proportional to the door — reversible not over-deliberated; irreversible not rushed.
- [ ] "What would change my mind" is stated as concrete, falsifiable evidence.
- [ ] An outside view was taken: a base rate / reference class was considered.
- [ ] At least three genuinely different alternatives were generated (incl. do-nothing/reversible test).
- [ ] The downside / worst case is named, and whether it is survivable is assessed.
- [ ] Expected value (rough probability x payoff) was estimated for the leading options.
- [ ] A premortem and/or inversion was done: at least one concrete failure mode identified and addressed.
- [ ] Key biases were explicitly checked: confirmation, sunk cost, anchoring, overconfidence.
- [ ] The decision and its reasoning (assumptions + expectations) are documented for later review.
- [ ] A calibration review is scheduled/noted, judging the *process* separately from the *outcome*.
- [ ] `personal/` recall has been consulted; any conflict with generic advice is flagged and personal/ wins.
