# Home

**New here?** Start with [[first run]] — a 10-minute guided setup of your personal layer.

Top-level entry point for the vault. Three spines:

## Capabilities

- refines:: [[Expertise Packs]] — everything the agent can help with, by domain.

## Me

- related:: [[personal/identity]] — identity, values, and goals. The personal layer
  (`vault/personal/`) holds one note per domain; see [[about the personal layer]].

## PARA (folders)

Organized by actionability, not topic:
- `vault/01-projects/` — active, deadline-bound efforts.
- `vault/02-areas/` — ongoing responsibilities to maintain.
- `vault/03-resources/` — reference material by topic.
- `vault/04-archive/` — inactive items.

## Flow

New thoughts come in via the `vault-capture` skill (or `scripts/capture.sh`);
periodic cleanup via `vault-review`; versioning via `vault-git`.
