# avatar-refs

The canonical avatar reference pack. Save the three images here (downloaded from
your Higgsfield history — see the job ids in `../AVATAR.md`):

- `portrait.png`    — face + style
- `fullbody.png`    — silhouette, outfit, legs
- `expressions.png` — 5 expressions (match these BlendShapes in VRoid)

These drive the VRM build; the final rigged avatar goes to `public/avatar.vrm`.
